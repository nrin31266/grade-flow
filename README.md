# GradeFlow

GradeFlow is a Vietnamese academic planning workspace for students.  
It helps manage study programs, real transcript records, GPA rules, graduation credits, and retake/improvement policies.

The project is built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui.

## Features

- Study program management
- Real transcript tracking by semester
- GPA 10/4 calculation
- Configurable school GPA rules
- Retake and improvement policy handling
- JSON import for study program data
- Local-first data storage
